URL Shortener

Usage

POST .../api/shorturl - stores the URL and responds JSON with the original URL and short_url code

GET .../api/shorturl/short_url - redirects to URL associated with short_url

Example

.../api/shorturl/rMj8NOTfdd redirects to https://freecodecamp.com
